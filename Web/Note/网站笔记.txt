- WordPress（WP）：全球最流行的 CMS 内容管理系统，用来建网站、博客、商城。
WordPress 是一个PHP 写的开源网站程序（CMS），它不能直接在你自己的电脑上跑给全世界看，必须放在一台24 小时开机、连公网的服务器上，这台服务器就是「WordPress 服务器」。


PHP是一种编程语言，底层也是用C语言封装的，和py一样 也是C语言套壳，然后用PHP解释器。



PHP = 做网页的 “后台逻辑”（处理数据、连接数据库、生成页面）
WordPress 就是 用 PHP 写出来的程序。
CMS = Content Management System（内容管理系统）
一句话：
CMS = 让你可以 “后台发文章、上传图片、管理网站” 的整套系统




- 云服务器提供的预装好 WordPress 的系统包，你选了就能直接创建一台 “已经装好 WordPress 的服务器”，不用自己装 PHP、MySQL、Nginx，一键建站。
- 英文：Visual Panel / Web-based Management Panel
- 就是一个网页图形界面（类似后台控制面板）。
让你不用敲 Linux 命令，就能：
管理网站
安装应用
配置域名
重启服务
备份
适合新手，不熟悉命令行也能轻松操作。
- 不需要自己写代码、配置环境，只需点一个按钮，系统自动帮你：
安装环境
下载程序
初始化网站
完成运行
适合快速搭建 WordPress、Discuz、网站模板等。
